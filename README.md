# GlobalRetail
This project is a complete, practical end-to-end data engineering workflow built on Databricks. It simulates how modern data teams design, ingest, process, store, and analyze large-scale datasets using industry-standard architectures and tools.


Tools Used: Databricks, Delta Lake, DatabricksSQL, PowerBI
